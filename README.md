# 🌿 Plant Disease Detection using YOLOv8

An AI-powered plant disease detection system that leverages the YOLOv8 object detection model to identify diseases in tomato leaves with high accuracy. This project aims to support precision agriculture by enabling fast and reliable disease diagnosis.

---

## 📌 Overview

Plant diseases significantly reduce agricultural productivity and crop quality. This project utilizes the YOLOv8 deep learning model to detect and classify diseases from tomato leaf images in real time.

The model is designed to assist farmers and researchers in early disease detection, reducing crop loss and promoting sustainable farming practices.

---

## ✨ Features

- Real-time tomato leaf disease detection
- YOLOv8 object detection model
- Image preprocessing and augmentation
- Model training and evaluation
- Performance analysis using Precision, Recall, F1-Score and mAP
- Suitable for deployment on edge devices
